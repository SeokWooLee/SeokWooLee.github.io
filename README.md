# iOS 개발 블로그

현직 iOS 개발자 SeokWoo Lee의 기술 블로그입니다.

- 주소: <https://seokwoolee.github.io>
- Swift, iOS 개발 경험과 소프트웨어 엔지니어링 원칙을 공유합니다.

## 구성

- [Jekyll](https://jekyllrb.com/) + [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/) 테마 (vendored)
- GitHub Pages로 호스팅 (master 브랜치 자동 빌드)

## 글 작성

`_posts/` 디렉토리에 `YYYY-MM-DD-제목.md` 형식으로 작성합니다.

```yaml
---
title: "글 제목"
description: "검색 결과와 AI 답변 엔진에 노출되는 요약문"
categories:
  - iOS
tags:
  - Swift
toc: true
toc_sticky: true
---
```

본문 헤딩은 `##`(H2)부터 시작합니다. 페이지 제목이 H1을 차지하기 때문입니다.
